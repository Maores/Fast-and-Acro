# Fast and Acro

A mobile game built with Unity, targeting Android first with iOS to follow.

## Tech Stack

- **Engine:** Unity 2022 LTS+
- **Render Pipeline:** Universal Render Pipeline (URP)
- **Language:** C#
- **Platforms:** Android (primary), iOS (secondary)

## Project Structure

```
Fast-and-Acro/
  Docs/       # Game design documents, technical specs
  Design/     # UI mockups, wireframes, flow diagrams
  Art/        # Source art files, concept art, references
  Audio/      # Source audio files (pre-import)
  Builds/     # Build outputs (gitignored)
  Tools/      # Editor tools, build scripts, utilities
  Scripts/    # Standalone scripts (build automation, CI/CD)
```

The Unity project folder will be created via Unity Hub when the project is initialized.

## Getting Started

1. Install Unity 2022 LTS or newer via Unity Hub
2. Clone this repository
3. Open the project in Unity Hub
4. Ensure URP is configured under Project Settings > Graphics
5. Open the main scene and press Play
